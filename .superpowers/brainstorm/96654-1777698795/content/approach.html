How should "How to Play" work?

Three approaches — pick the one that fits how new players will actually use this

A

Richer static section (in-place)

Expand the existing rulesBox into a multi-section card on the home screen — sections for Scoring, Trump, and Bidding with icons. Always visible, no interaction required.

Pros

  • Zero friction — just scroll down
  • No new screen or state
  • Minimal code change

Cons

  • Home screen gets longer
  • Can't show full rules depth
  • Veteran players always see it
B

Expandable card (accordion)

The current rulesBox becomes tappable — tap "How to Play" to expand it in-place with full rules (5–6 sections). Collapsed by default, so veterans aren't bothered.

Pros

  • Progressive disclosure
  • Stays on home screen, no modal
  • Clean collapsed state

Cons

  • Expands pushes buttons up/off-screen
  • Slightly more complex than static
C

Modal / bottom sheet

A "How to Play" link/button opens a full-screen modal with all rules organized by phase (Setup → Bidding → Playing → Scoring). Scrollable, dismissible with a close button.

Pros

  • Unlimited space for rules depth
  • Home screen stays clean
  • Can include examples and scoring table

Cons

  • Extra tap to access
  • More code (Modal component)

My recommendation: B (expandable) — home screen stays clean for veterans, new players get full rules without leaving the screen. Modal (C) is worth it only if the rules need more depth than an expandable allows.
